# OpenCSV - Product Requirements Document

**Version:** 1.0  
**Date:** 2026-05-27  
**Product:** OpenCSV — 开源 CSV 编辑器（SmoothCSV 平替方案）

---

## 1. 产品愿景

OpenCSV 是一款面向数据工作者的高性能 CSV 编辑器，目标是成为"表格数据的 VS Code"：

- **开源免费**：MIT 协议，完全开放
- **高性能**：100MB 大文件秒开，支持百万行流式渲染
- **功能完备**：媲美 SmoothCSV 的全套数据处理能力
- **本地优先**：数据不上云，Go 后端本地运行，Vue 前端嵌入交付

## 4. 非功能需求

### 4.1 性能指标

| 指标 | 目标 |
|------|------|
| 100MB CSV 打开时间 | < 3 秒（首屏数据可见）|
| 1M 行滚动帧率 | > 30 FPS |
| 查找替换 1M 行 | < 1 秒 |
| SQL 查询 100K 行 | < 500ms |

### 4.2 可靠性

- 文件保存前自动备份
- 操作崩溃不丢失未保存数据（LocalStorage 草稿）
- 大文件分块流式读取，防止内存溢出

### 4.3 安全性

- 纯本地运行，不发送任何数据到远程
- 本地 API 仅监听 localhost

## 5. 里程碑计划

| 里程碑 | 内容 | 验收标准 |
|--------|------|---------|
| **M1** 核心基础 | 项目框架、文件打开/保存、虚拟网格、基础编辑、撤销重做 | 能打开并编辑 CSV，保存后数据一致 |
